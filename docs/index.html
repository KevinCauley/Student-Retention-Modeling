Project Overview

This project analyzes undergraduate student data from Advance University to better understand the factors associated with academic performance and retention risk. The primary modeling outcome is final course grade, with enrollment status used as a predictor rather than the outcome.

The project is framed around an early-support workflow. Instead of waiting until a student stops out, the analysis looks for academic, attendance, financial, support-service, and engagement patterns that may indicate a need for follow-up.

Core Focus

Final grade prediction, academic-risk identification, model comparison, and student-support planning.

Why This Project Matters

Student retention is not only an enrollment outcome. It is also an early-warning challenge. Academic performance, attendance, financial pressure, engagement, and support-service usage can all provide signals that a student may need help before they disengage.

Outcome

A practical analytics workflow for predicting student academic performance, identifying risk patterns, and recommending targeted retention-support actions.
